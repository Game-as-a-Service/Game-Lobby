# Game-Lobby

The frontend of the Game as a Service lobby.

## Commands

```bash
# Host by Yarn
1. yarn
2. yarn dev

# Host by Docker
1. Install Docker and Docker compose
2. docker compose build
3. docker compose up

# Lint
yarn pre-commit

# Runs All Test
yarn test

## Unit Test
yarn unit-test
### Test singel files/folder
yarn jest $PATH

## E2E Test
yarn e2e-test

# Storybook
yarn storybook
```

## Git Cooperation

- Branch Name: `feature/ticket-name`
  - `git checkout -b feature/[ticket-name]`
- Commit: `feat: ticket-name`

## Naming Convention

- Route: snake-case
- Components: PascalCase

## Components DoD

1. Test
2. Implementation
3. Doc(Storybook)
